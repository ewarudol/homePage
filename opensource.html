Ulubiony open-source

i alternatywy technologiczne


Messenger/Whatsapp -> Signal

  • Komunikator, który nie podgląda.

Gmail -> Protonmail

  • Skrzynka, która nie podgląda.

Google -> DuckDuckGo

  • Wyszukiwarka, która nie ogranicza widoczności niewygodnych wyników (i nie podgląda).

Facebook -> Feedly

  • Żródło codziennych informacji, które nie pożera twojego czasu (...i nie podgląda).

Windows -> Linux Manjaro

  • System operacyjny bez zbierania danych i wirusów.

Youtube -> Freetube(desktop)/Invidious(przeglądarka)/NewPipe(android)

  • Youtube bez zbierania danych.

Chrome -> Brave/Firefox

  • Przeglądarka blokująca śledzenie.

Epicgames -> GOG

  • Sklep z grami bez DRMów (na zawsze twoje).

Redirector

  • Wtyczka do przeglądarki, żeby nie wchodzić na wścibskie strony.

Keepass

  • Bezpieczna baza haseł.

MS Office -> Free Office

  • Pakiet biurowy za darmo, na zawsze, bez abonamentu.

SurfShark

  • VPN, żeby nie ogłaszać gdzie się znajdujesz.

Adblock/Blockera + Pi hole

  • Wtyczka do przeglądarki żeby nie oglądać reklam, filtr reklam.

Visual Studio -> Jet Brains Rider

  • .NETowe IDE na Linuxa
